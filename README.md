🌿 Live Site: https://justishmael.github.io/

## About

I’m an autistic developer and data analyst interested in building calm, accessible systems for people and communities.

My work focuses on:

- Kotlin development
- Data analysis
- AI-assisted tooling
- Human-centered software
- Accessibility-conscious design
- Reducing cognitive friction

I’m also interested in indie game development, voxel systems, and sandbox spaces like Minecraft that encourage creativity, collaboration, and building joy.

---

## Tech Stack

- Kotlin
- Compose Multiplatform
- Compose Web
- GitHub Pages
- GitHub Actions

---

## Development

Run locally:

```bash
./gradlew jsBrowserDevelopmentRun
```

Build production distribution:

```bash
./gradlew jsBrowserDistribution
```

---

## Deployment

This site deploys automatically to GitHub Pages through GitHub Actions whenever changes are pushed to `main`.
